# Images des motos

## Télécharger les vraies images Yamaha

Pour remplacer les placeholders par les vraies images:

1. Allez sur yamaha-motor.eu pour chaque modèle
2. Téléchargez l'image principale
3. Renommez: xsr700.jpg, tracer7.jpg, mt07.jpg, tenere700.jpg, tracer9gt.jpg, mt09.jpg
4. Placez dans ce dossier

Ensuite exécutez depuis la racine du projet:
node update-local-images.js
